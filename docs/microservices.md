# Описание микросервисной архитектуры

## HTTP

Основной микросервис, отвечающий за обработку HTTP-запросов от клиента.

Вызывает сервисы Auth и Search

## Auth

Микросервис, отвечающий за аутентификацию и авторизацию пользователей.

Создает и валидирует jwt токены.

Хранит refresh токены в базе данных.

## Search

Микросервис, отвечающий за поиск и обработку запросов поиска.
