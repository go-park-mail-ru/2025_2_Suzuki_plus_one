# MinIO

Мы используем MinIO в качестве решения для хранения медиафайлов (аватары пользователей, превьюшки фильмов, сами фильмы и т.д.).

> Если сильно надо можно рассмотреть переезд на TimeWeb Object Storage или другой облачный s3 сервис.

## Бакеты

- avatars
- posters
- trailers
- medias (фильмы и сериалы)
